# sed

`sed` is a stream editor, which can transform texts.
It is designed as a command line data processor.

<!--more-->

Example:

```
sed -e 's/^[[:space:]]*//'
```

This removes all spaces from a string.

| distribution | package name   |
| ------------ | -------------- |
| ubuntu       | `sed`          |
| gentoo       | `sys-apps/sed` |
